* 트윗 추천 알고리즘
- 대부분 scala(= 자바 대용품)로 개발 함
--> 자바 대용품이니 자바와 호환성도 좋음
=> 메인 피드에 뿌리는 방법
1) 1500개의 정보를 db에서 가져옴
2) 이 중에 절반은 내가 팔로잉 하는 계정의 트윗
    나머지 절반은 팔로잉 안하는 계정
     => 포함: 팔로잉 한 사람이 좋아요한거 / 나랑 유사한 사람이 좋아요 한거(클러스터링으로 분류)
3) 관심사 별 계정 구분지어 그룹 생성(= community)
 => 여기서도 트윗 뽑아옴
4) 가져온 모든 트윗들을 딥러닝 모델에 넣고 트윗들마다 점수를 매김
 => 딥러닝 모델 == 참여가 높은 트윗들에 높은 점수 부여해줌
5) 1500개 중 점수가 높은 거 부터 보여줌
 => 필터링 과정 거침: 개인적으로 차단한 것들 / 중복내용 / 비관심사 등 제거
